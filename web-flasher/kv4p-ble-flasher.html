KV4P HT BLE Flasher

Flash a KV4P HT ESP32 with firmware that keeps the normal USB KISS protocol and adds the Nordic UART BLE transport required by KV4P/ATL.

Use Chrome, Edge, Brave, or another Chromium browser on macOS, Windows, Linux, or ChromeOS. Connect the radio ESP32 by USB, then choose what to flash.

Build Latest

The static page flashes the generated binary. To rebuild from latest upstream KV4P source, run:

firmware/ble_bridge/build_ble_release.sh --update

That script applies only the BLE overlay and emits firmware/manifest-ble-latest.json for this page.
