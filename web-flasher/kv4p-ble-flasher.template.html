KV4P HT BLE Flasher

Manual BOOT flashing steps

  1. Use Chrome, Edge, Brave, or another Chromium browser on macOS, Windows, Linux, or ChromeOS.
  2. Close the KV4P/ATL app, serial monitors, and anything else using the radio USB serial port.
  3. Plug the KV4P HT ESP32 into this computer by USB-C or USB.
  4. Press and keep holding the radio board's BOOT button before starting the flash.
  5. Click Start manual BOOT flash, confirm the BOOT reminder, then choose the CP2102/USB serial port in the browser picker.
  6. Keep holding BOOT until the top banner says You may now release the BOOT button. The large ESP Web Tools install box is normal.
  7. When flashing finishes, disconnect and reconnect USB or tap RST once, then use Scan for KV4P BLE.

Regenerate

Every firmware release should regenerate this HTML so the newest firmware is embedded:

firmware/ble_bridge/build_ble_release.sh --update

The script applies the BLE overlay, builds firmware, writes the normal manifest/bin files, then rebuilds this all-in-one HTML file.

Hold BOOT before choosing the port

The ESP32 must be in serial download mode before ESP Web Tools can initialize it. Keep the BOOT button held through the next browser serial picker.

  1. Press and hold BOOT now.
  2. While still holding BOOT, click I am holding BOOT - start flash.
  3. Choose the KV4P HT CP2102/USB serial port in the browser picker.
  4. Do not release BOOT while the flasher says connecting, preparing, or initializing.
  5. Release BOOT only when the top banner says You may now release the BOOT button.

If initialization still fails, release BOOT, tap RST once or reconnect USB, then retry from the first BOOT step.
